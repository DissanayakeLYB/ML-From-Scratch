# ML From Scratch

Learning Machine Learning fundamentals by implementing core algorithms from scratch in Python.

## Goals
- Understand ML math and intuition (not just libraries)
- Implement algorithms using pure Python
- Build models step-by-step and evaluate them properly
- Gradually transition to NumPy and scikit-learn

## Current Progress
- [x] Linear Regression (from scratch)
- [ ] Logistic Regression
- [ ] Gradient Descent
- [ ] Regularization
- [ ] Neural Networks

## Structure
Each folder contains:
- data loading
- model implementation
- training logic
- evaluation metrics

No magic. No black boxes.
